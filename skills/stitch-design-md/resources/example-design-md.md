# Design System Name: TravelFlow

## Color Palette

- `--primary`: 210 100% 50% (Sky Blue)
- `--secondary`: 150 60% 45% (Jungle Green)
- `--background`: 0 0% 100% (Pure White)
- `--surface`: 210 20% 98% (Light Gray)

## Typography

- **Headings**: Inter, Sans-serif, Bold
- **Body**: Roboto, Sans-serif, Regular

## Spacing

- `8px` increments (Grid: 4, 8, 16, 24, 32, 48, 64)

## Component Rules

### Buttons

- Rounded corners: `12px`
- Shadow: `subtle-elevation`
- Hover: Darken by 10%
