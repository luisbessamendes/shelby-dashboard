# Generating Videos with Remotion

## When to use this skill

- When a user wants to present their Stitch designs in a professional video format.
- When creating marketing assets or product demos for a Stitch project.
- When you need to showcase the "flow" between multiple Stitch screens.

## Workflow

1. **Identify Screens**: Decide which Stitch screens to feature in the video.
2. **Setup Remotion**: Create a new Remotion project or use an existing template.
3. **Download Assets**: Retrieve screen images/screenshots from the Stitch project.
4. **Compose Video**:
    - Add smooth transitions (fades, slides) between screens.
    - Add zooming effects to highlight specific UI components.
    - Overlay descriptive text or voiceover scripts.
5. **Render**: Run the Remotion render command and provide the output to the user.

## Instructions

- Use **panning and zooming** to make static screenshots feel dynamic.
- Match the **video theme** to the design system in `DESIGN.md`.
- Ensure all screens are high-resolution for the best render quality.
- Keep videos concise (typically 30-60 seconds for a showcase).
