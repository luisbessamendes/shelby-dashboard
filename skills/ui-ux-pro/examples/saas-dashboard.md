# Example: SaaS Dashboard Design System

**Project**: CloudMetrics Pro
**Product Type**: B2B Analytics SaaS

## Core Tokens

- **Primary**: HSL(220, 90%, 56%)
- **Background**: HSL(224, 71%, 4%)
- **Surface**: HSL(222, 47%, 11%)
- **Typography**: Inter (Headings), JetBrains Mono (Data)

## Visual Pattern: "Bento Analytics"

- High-elevation cards with subtle 1px borders.
- Top-level summaries in 3-column grid.
- Detail charts in spanning rows.

## UX Guidelines

- **Zero Data State**: Show skeletal loaders, not empty white space.
- **Filtering**: Instant updates using local state or optimistic UI.
- **Focus**: Maintain `ring-primary` on all keyboard-navigable elements.
