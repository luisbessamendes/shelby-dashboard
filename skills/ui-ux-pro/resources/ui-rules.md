# Professional UI/UX Rules

### Icons & Visuals

| Rule | Do | Don't |
|------|----|-------|
| Icon Type | Use SVG (Lucide, Heroicons) | Use Emojis (🚀, ⚙️) |
| Branding | Use Simple Icons SVGs | Guess logo paths |
| Sizing | Consistent 24x24 (w-6 h-6) | Random sizing |

### Interactivity

| Rule | Do | Don't |
|------|----|-------|
| Cursors | `cursor-pointer` on all cards | Default cursor on buttons |
| Feedback | Color/Shadow/Border change | No visual feedback |
| Motion | `transition-colors duration-200` | Instant jumps or >500ms lag |

### Contrast & Hierarchy

| Rule | Do | Don't |
|------|----|-------|
| Text Color | Slate-900 (High contrast) | Light gray on white |
| Card Style | Multi-layer shadow | Flat borders only |
| Glass-morphism | High opacity (80%+) for light | Transparent (10%) for light |
